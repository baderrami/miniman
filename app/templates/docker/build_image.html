Build an Image from a Dockerfile
Enter the path to the directory containing the Dockerfile.
Enter a tag for the image (name:tag format).
Notes:
  • The directory must contain a file named Dockerfile.
  • The build context will be the directory containing the Dockerfile.
  • Building an image may take some time depending on the complexity of the Dockerfile.
  • Make sure the user running the application has permission to access the Dockerfile path.
Example Dockerfile:
FROM nginx:alpine
COPY ./html /usr/share/nginx/html
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
